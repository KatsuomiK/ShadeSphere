ShadeSphere
===========
A script to create a sphere shade which prevents users from looking back.

# Requirements

- Oculus Rift DK2
- Oculus Runtime 0.6.0+
- Unity 5.1.1+

# How to use

Drop ShadeShpere prefab to your scene and play.
